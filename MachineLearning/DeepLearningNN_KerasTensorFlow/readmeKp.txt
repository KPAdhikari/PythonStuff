Used https://www.tutorialspoint.com/keras/keras_installation.htm for Keras installation in the virtual environment named 'kerasenv'
with 'python3 -m venv kerasenv'
Step2: cd to 'kerasvenv' and run 'source bin/activate'
Keras depends on the following python libraries: Numpy, Pandas, Scikit-learn, Matplotlib, Scipy, Seaborn. So, installed each of them first with 'pip install numpy' etc.
